### Админка:
```
/board/login        - Вход
/board              - Главная админки
/board/experts      - Управление экспертами
/board/events       - Управление событиями
/board/appointments - Заявки на консультации
/board/services     - Услуги (в разработке)
/board/reviews      - Отзывы (в разработке)
/board/settings     - Настройки (в разработке)
```

### API Endpoints:

**Публичные:**
- `POST /api/contact` - Контактная форма
- `GET /api/experts/[id]` - Получить эксперта
- `GET /api/experts/[id]/slots` - Получить слоты эксперта
- `POST /api/appointments` - Запись на консультацию
- `GET /api/events/[id]` - Получить событие
- `POST /api/events/[id]/register` - Регистрация на событие

**Авторизация:**
- `POST /api/auth/login` - Вход
- `POST /api/auth/logout` - Выход
- `GET /api/auth/check` - Проверка авторизации

**Админка (требуется авторизация):**
- `GET /api/board/experts` - Список экспертов
- `POST /api/board/experts` - Создать эксперта
- `PUT /api/board/experts/[id]` - Обновить эксперта
- `DELETE /api/board/experts/[id]` - Удалить эксперта
- `GET /api/board/events` - Список событий
- `POST /api/board/events` - Создать событие
- `PUT /api/board/events/[id]` - Обновить событие
- `DELETE /api/board/events/[id]` - Удалить событие
- `GET /api/board/appointments` - Список заявок
- `PUT /api/board/appointments/[id]` - Обновить статус заявки

## 📊 БАЗА ДАННЫХ:

### Модели:
- **Expert** - эксперты/коллеги
- **ExpertSlot** - временные слоты для записи
- **Event** - события/афиша
- **EventRegistration** - регистрации на события
- **Appointment** - заявки на консультации
- **ContactForm** - контактные формы
- **Admin** - администраторы
- **Service, Course, Test, Book, Review, SiteSettings** - остальные модели

### Команды Prisma:
```bash
npx prisma studio          # Открыть GUI для просмотра БД
npx prisma migrate dev     # Создать миграцию
npx prisma generate        # Сгенерировать Prisma Client
```

## 🚀 ДЕПЛОЙ:

### Для деплоя на timeweb.cloud:
1. Соберите проект: `npm run build`
2. Загрузите файлы на сервер
3. Настройте переменные окружения
4. Запустите: `npm start`

### Переменные окружения:
```env
DATABASE_URL="file:./dev.db"
NEXTAUTH_SECRET="your-secret-key"
NEXTAUTH_URL="https://olgamagdenko.ru"
```
