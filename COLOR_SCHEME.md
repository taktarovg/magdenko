# 🎨 ОБНОВЛЕННАЯ ЦВЕТОВАЯ СХЕМА

## Новые цвета (теплая, дружелюбная палитра):

### Основные цвета:
- **Primary (Основной)**: `#C87E5D` - Теплый терракотовый
  - Используется для: кнопки, ссылки, акценты, заголовки
  - Психология: теплота, доверие, профессионализм
  
- **Secondary (Вторичный)**: `#E8B86D` - Мягкий золотистый
  - Используется для: дополнительные акценты, иконки
  - Психология: оптимизм, энергия, позитив

- **Accent (Акцент)**: `#D4A574` - Бежево-золотой
  - Используется для: hover-эффекты, подсветка
  - Психология: спокойствие, гармония

### Фоновые цвета:
- **Background**: `#FFF9F5` - Очень светлый кремовый
  - Мягкий теплый фон вместо холодного белого
  
- **Muted**: `#FFF4ED` - Светло-персиковый
  - Для карточек и выделенных блоков

### Текст:
- **Foreground**: `#3D2E2E` - Темно-коричневый
  - Вместо холодного серого - теплый коричневый

- **Muted Foreground**: `#8B7355` - Средне-коричневый
  - Для вторичного текста

### Границы:
- **Border**: `#F2E4D8` - Бежевый
  - Мягкие границы вместо холодных серых

---

## Сравнение с предыдущей схемой:

### Было (холодная):
```css
Primary: #2B5A7E    /* Холодный синий */
Secondary: #E8B86D  /* Золотой */
Accent: #7FB3D5     /* Холодный голубой */
Background: #F8F9FA /* Серо-белый */
Foreground: #2C3E50 /* Холодный серый */
```

### Стало (теплая):
```css
Primary: #C87E5D    /* Теплый терракотовый */
Secondary: #E8B86D  /* Золотой (оставили) */
Accent: #D4A574     /* Теплый бежево-золотой */
Background: #FFF9F5 /* Кремовый */
Foreground: #3D2E2E /* Теплый коричневый */
```

## Психологическое воздействие:

### Холодная гамма (было):
- ❄️ Профессионализм, но отстраненность
- 🔷 Доверие, но дистанция
- 🧊 Стабильность, но холодность

### Теплая гамма (стало):
- 🤗 Тепло и забота
- 🌅 Уют и комфорт
- 💛 Эмпатия и понимание
- 🏡 Безопасность и доверие

---

## Ширина контента:

### Было:
- Container: до 1536px (очень широко)

### Стало:
- Container: максимум 1200px
- Более комфортное чтение
- Контент не "растекается" на больших экранах

---

## Применение:

После обновления `globals.css` новые цвета автоматически применяются ко всему сайту через CSS-переменные и Tailwind классы:

- `bg-primary` → теплый терракотовый
- `text-primary` → теплый терракотовый
- `bg-background` → кремовый фон
- `border-border` → бежевые границы
- `text-foreground` → теплый коричневый текст

Перезапустите сервер, чтобы увидеть изменения! 🎨
